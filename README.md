# Digital Image Processing (Python)

This repository contains a collection of Python scripts for Digital Image Processing tasks.

## Topics Covered
- Image intensity and RGB processing
- Brightness and contrast adjustment
- Grayscale and negative images
- Image transformation (cropping, rotating, scaling)
- Edge detection (Sobel, Prewitt, Robert, Laplacian)
- Noise reduction and sharpening
- Image compression techniques

## Tech Stack
- Python
- OpenCV
- NumPy

This project was created as part of Digital Image Processing practice.
